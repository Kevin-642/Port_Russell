Exemple de requête curl pour créer un catway

curl -X POST https://example.com/api/catways/create \
-H "Content-Type: application/json" \
-d '{
  "catwayNumber": 1,
  "type": "long",
  "catwayState": "Disponible"
}'

Exemple de réponse en JSON

{
  "_id": "60f5a3e2cdae1f0012fca9a2",
  "catwayNumber": 1,
  "type": "long",
  "catwayState": "Disponible"
}
